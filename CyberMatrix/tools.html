Breach Response Mini Guide

// quick response steps for common security incidents

I clicked a bad link

  1. Disconnect from risky pages and close the tab.
  2. Do not enter passwords or codes.
  3. Change the affected account password from a trusted site.
  4. Enable MFA and review active sessions.

My account was hacked

  1. Reset the password from a clean device.
  2. Remove unknown recovery emails, devices, and sessions.
  3. Check inbox rules and recent activity.
  4. Tell contacts to ignore suspicious messages.

I downloaded malware

  1. Disconnect from Wi-Fi or unplug ethernet.
  2. Do not open the downloaded file again.
  3. Run a full scan with updated security software.
  4. Preserve file name, time, and source URL for reporting.
